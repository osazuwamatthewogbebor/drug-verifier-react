# SafeVerify - Drug Authenticity Portal

SafeVerify is a modern, premium pharmaceutical verification platform designed to combat the circulation of counterfeit medication globally. Built with a focus on trust, security, and medical-grade aesthetics.

![SafeVerify Logo](https://raw.githubusercontent.com/lucide-react/lucide/main/icons/shield-check.svg)

## 🚀 Key Features

- **Instant Verification**: Real-time authenticity checks using a 12-digit security code.
- **Global Network**: Integration with certified manufacturers and international diagnostic labs.
- **Safety Alerts**: Real-time tracking of drug recalls and security advisories from global health organizations.
- **Mobile First**: Fully responsive design for patients and healthcare providers on the go.
- **Privacy Centric**: Anonymous verification history stored locally to protect patient privacy.

## 🛠️ Tech Stack

- **Framework**: React 18
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion
- **Tooling**: Vite

## 📥 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/osazuwamatthewogbebor/drug-verifier-react.git
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run development server**:
   ```bash
   npm run dev
   ```

4. **Build for production**:
   ```bash
   npm run build
   ```

## 🔐 Security Standards

SafeVerify utilizes a distributed hash registry to verify medications without storing personal patient data. For more information, visit our [Privacy Policy](src/pages/Privacy.jsx) or [Terms of Service](src/pages/Terms.jsx).
