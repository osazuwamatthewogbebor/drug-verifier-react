**SafeVerify** is a premium, high-integrity React application designed to combat the global circulation of counterfeit medication. It provides patients, healthcare providers, and manufacturers with a seamless, encrypted portal to verify medication authenticity in real-time.

---

## ✨ Key Features

- **🔍 Instant Verification**: 12-digit security code authentication against a global pharmaceutical registry.
- **📊 Security Dashboard**: Real-time stats on hospital trust, verification volume, and global compliance.
- **📜 Verification History**: Secure, local-only audit logs of all your authenticity checks.
- **⚠️ Safety News Hub**: Live alerts for drug recalls, counterfeit detections, and security updates.
- **🧪 Lab Network**: Integrated directory of certified diagnostic laboratories for physical analysis.
- **🚨 Incident Reporting**: Comprehensive, multi-step reporting portal for suspicious medication.

## 🎨 Design Philosophy

Built with a **Premium Healthcare Aesthetic**:
- **Glassmorphism**: Subtle, translucent UI elements for a modern feel.
- **Micro-interactions**: Smooth Framer Motion animations for high-quality UX.
- **Typography**: Industrial-grade font pairing (Inter & Manrope) for maximum readability.
- **Security-First**: Visual cues and encrypted data protocols used throughout the platform.

## 🚀 Tech Stack

- **Framework**: React 18+ (Hooks, Router)
- **Styling**: TailwindCSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Persistence**: LocalStorage API

## 🛠️ Getting Started

### Prerequisites
- Node.js (v16.0 or higher)
- npm or yarn

### Installation

1.  **Clone the repository**
    ```bash
    git clone https://github.com/osazuwamatthewogbebor/drug-verifier-react.git
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Start the development server**
    ```bash
    npm run dev
    ```
